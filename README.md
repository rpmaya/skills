# Agent Skills · Guía Avanzada 2026 — Curso ML2

Guía didáctica y completa sobre **Agent Skills**, las capacidades modulares de Claude (Anthropic) que extienden la funcionalidad del agente mediante *progressive disclosure*. Material de apoyo del curso **Aprendizaje Automático II** (ML2).

## Contenido

Un único HTML autocontenido (`index.html`) con teoría, ejemplos y novedades 2026:

- **Fundamentos** — Qué son las Skills, arquitectura *progressive disclosure*, anatomía del `SKILL.md`
- **Configuración avanzada** — Multi-file skills, control de invocación, buenas prácticas de tamaño
- **Skills vs. otras opciones** — Cuándo usar Skills, prompts, MCP o fine-tuning
- **Distribución** — Vía Git, plugins, marketplaces y Managed Settings
- **Troubleshooting y seguridad** — Activación, prioridad, runtime y consideraciones de riesgo
- **Novedades 2026** — Estándar abierto `agentskills.io`, Dreaming, Outcomes, Multi-Agent Orchestration, Managed Agents, Add-ins, Plugins SDK, Enterprise, Python SDK
- **Novedades junio 2026** — `/reload-skills`, hook `reloadSkills: true`, auto-load de `.claude/skills/`, `disallowed-tools` en frontmatter, `defaultEnabled: false`, Claude Opus 4.8, facturación separada del Agent SDK
- **Roadmap y cheatsheet**
